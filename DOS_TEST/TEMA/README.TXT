Remoute Debugger Server для Delphi 5.0 Enterprise
-----------------------------------------------------
Для установки Remoute Debugger Server вам нужно
нажать кнопку Установить. Тот же эффект будет
достигнут двойным щелчком кнопки мыши на файле
SETUP.EXE. Можно просто выбрать этот файл и нажать
клавишу ENTER.
-----------------------------------------------------
Успешной работы !
